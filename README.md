# 🎂 Sara's Birthday Website 🎂

A beautiful, romantic birthday website created with love for **Sara**! 💕

## 🌟 Features

✨ **Beautiful Romantic Design**
- Pink and Red romantic color theme
- Smooth animations and transitions
- Responsive design for all devices

⏱️ **Countdown Timer**
- Live countdown to 16th May
- Shows Days, Hours, Minutes, and Seconds
- Auto-updates every second

📸 **Photo Gallery**
- Display beautiful memories
- Smooth hover effects
- Responsive grid layout

🎵 **Birthday Song**
- Embedded music player
- Play "Sang Rahiyo" 
- Full audio controls

💌 **Love Message Section**
- Romantic message for Sara
- Beautifully styled text
- Easy to customize

📱 **QR Code**
- Shareable QR code
- Scans to website URL
- Perfect for sharing on social media

## 🚀 How to Use

1. **Access the Website**: Visit `https://vijayramdasrathod-droid.github.io/sara-birthday`

2. **Customize Love Message**: 
   - Edit `index.html`
   - Find the love message section
   - Replace with your personal message

3. **Add Photos**:
   - Replace `photo1.jpg`, `photo2.jpg`, `photo3.jpg` with actual images
   - Or update the image paths in `index.html`

4. **Update Music**:
   - Change the song URL in `index.html` music section if needed

## 🎨 Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #ff1493;
    --secondary-color: #ff69b4;
    --accent-color: #c71585;
}
```

### Update Love Message
Edit the message text in `index.html` in the love-message-section

### Add Photos
Upload your photos to the repository and update the image paths in `index.html`
